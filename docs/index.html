agentish.

A visual framework for building autonomous AI agents.

Agentish is a node-based environment for designing, testing, and deploying autonomous agent pipelines. Wire together LLM calls, tools, and decision logic in an intuitive graph editor — then run them against real challenges.

Features

  • Visual node-based graph editor for agent pipelines
  • First-class MCP (Model Context Protocol) tool integration
  • Real-time execution and state inspection
  • Export & import agent specifications as JSON
  • Built for security challenges and autonomous research

Quick Start

# Clone the repository
$ git clone https://github.com/shellphish/agentish.git
$ cd agentish
# Install dependencies
$ pip install -r requirements.txt
# Start the server
$ bash start_server.sh
